# Практичні завдання

Тема: **Hugging Face Transformers: готові pipeline, класифікація тексту і генерація**

## Перед початком

Працюйте в notebook `lesson_14_transformers_practice.ipynb`.

Для цього заняття достатньо малих відкритих моделей, які можна запустити на звичайному ПК:

- `distilbert-base-uncased-finetuned-sst-2-english` - класифікація настрою тексту;
- `gpt2` або `distilgpt2` - коротка генерація тексту;
- `sshleifer/tiny-gpt2` - дуже мала тестова модель, якщо ПК працює повільно або модель треба завантажити швидко.

Приклад базового імпорту:

```python
from transformers import pipeline
```

Якщо модель генерує англійський текст краще за український, використовуйте англійські приклади. Мета практики - навчитися запускати `pipeline`, змінювати параметри і аналізувати результат.

### Завдання 3. Генератор коротких product descriptions

Запустіть малу GPT-подібну модель і згенеруйте короткі описи товарів.

```python

    "text-generation",
    model="gpt2"
)
```

Якщо модель працює повільно, замініть `gpt2` на `distilgpt2` або `sshleifer/tiny-gpt2`.


3. Для кожного prompt-а згенеруйте текст із параметрами:

```python
result = generator(
    prompt,
    max_new_tokens=40,
    do_sample=True,
    temperature=0.8,
    top_p=0.9,
    pad_token_id=generator.tokenizer.eos_token_id
```
