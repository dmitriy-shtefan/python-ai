# Практика 3. MLP для класифікації вікової групи abalone

## Датасет

Використовується датасет **Abalone** з UCI Machine Learning Repository.

Джерело: https://archive.ics.uci.edu/dataset/1/abalone

У датасеті є 4177 молюсків abalone. Для кожного зразка доступні фізичні вимірювання: довжина, діаметр, висота, кілька типів ваги, стать і кількість кілець.

Цільова змінна: `rings`. Кількість кілець використовується як приблизний показник віку.

Щоб зробити задачу зручною для класифікації, перетворіть `rings` на 3 класи:

- `0` - молодші: `rings <= 8`;
- `1` - середні: `9 <= rings <= 10`;
- `2` - старші: `rings >= 11`.

## Бізнес-контекст

Уявіть, що дослідницька лабораторія хоче швидко оцінювати вікову групу abalone за зовнішніми вимірюваннями. Точний підрахунок кілець потребує ручної роботи, а модель може дати швидку попередню оцінку.

## Що потрібно зробити

### 1. Підготовка даних

У notebook:

1. Завантажте CSV у `pandas`.
2. Перевірте розмір датасету, типи колонок і розподіл `rings`.
3. Створіть нову цільову змінну `age_group` з трьома класами.
4. Закодуйте колонку `sex` через one-hot encoding.
5. Розділіть дані на `X` та `y`.
6. Зробіть `train/test split` у пропорції `80/20` з `random_state=42`.
7. Масштабуйте ознаки через `StandardScaler`.

### 3. PyTorch-тензори

Перетворіть дані на PyTorch-тензори:

- `X_train`, `X_test` -> `torch.float32`;
- `y_train`, `y_test` -> `torch.long`.

### 4. MLP через nn.Sequential

Побудуйте модель:

```text
Linear(input_dim -> 24)
ReLU
Linear(24 -> 12)
ReLU
Linear(12 -> 3)
```

`input_dim` визначте автоматично після one-hot encoding.

Для задачі використовуйте:

- функцію втрат `nn.CrossEntropyLoss`;
- змінну з функцією втрат назвіть `loss_fn`;
- оптимізатор `torch.optim.Adam`;
- `learning_rate=0.001`;
- 120-180 епох.

### 5. Train loop без батчів

Реалізуйте базовий train loop:

1. `model.train()`;
2. forward pass для всього `X_train_tensor`;
3. loss для всього `y_train_tensor`;
4. `optimizer.zero_grad()`;
5. `loss.backward()`;
6. `optimizer.step()`;
7. збереження train loss за епоху.

### 6. Оцінка якості

На test set порахуйте:

- `accuracy`;
- `f1_score` з `average="weighted"`;
- `confusion_matrix`.

Додайте короткий висновок:

- які вікові групи модель плутає найчастіше;
- чи підходять ці ознаки для точного визначення вікової групи;
- чи достатньо стабільно зменшується train loss.
