# Практика 1. MLP для класифікації сортів рису

## Датасет

Використовується датасет **Rice (Cammeo and Osmancik)** з UCI Machine Learning Repository.

Джерело: https://archive.ics.uci.edu/dataset/545/rice%2Bcammeo%2Band%2Bosmancik

У датасеті є 3810 зерен рису двох сортів:

- `Cammeo`;
- `Osmancik`.

Для кожного зерна доступні 7 числових морфологічних ознак, отриманих після обробки зображень:

- `Area`;
- `Perimeter`;
- `Major_Axis_Length`;
- `Minor_Axis_Length`;
- `Eccentricity`;
- `Convex_Area`;
- `Extent`.

Цільова змінна: `Class`.

## Бізнес-контекст

Уявіть, що виробник круп хоче автоматично розрізняти два сорти рису за вимірюваннями зерен. Замість ручної перевірки кожної партії компанія хоче навчити модель, яка за 7 числовими ознаками прогнозує сорт зерна.

Ваше завдання - побудувати MLP-класифікатор у PyTorch через `nn.Sequential` і навчити його за допомогою базового train loop.

## Що потрібно зробити


### 1. Підготовка даних

У notebook:

1. Завантажте `rice_cammeo_osmancik.csv` CSV у `pandas`.
2. Перевірте розмір датасету, назви колонок і баланс класів.
3. Розділіть дані на `X` та `y`.
4. Закодуйте цільову змінну:
   - `Cammeo` -> `0`;
   - `Osmancik` -> `1`.
5. Зробіть `train/test split` у пропорції `80/20` з `random_state=42`.
6. Масштабуйте ознаки через `StandardScaler`.

### 2. PyTorch-тензори

Перетворіть дані на PyTorch-тензори:

- `X_train`, `X_test` -> `torch.float32`;
- `y_train`, `y_test` -> `torch.long`.

### 3. MLP через nn.Sequential

Побудуйте модель:

```text
Linear(7 -> 16)
ReLU
Linear(16 -> 8)
ReLU
Linear(8 -> 2)
```

Для задачі використовуйте:

- функцію втрат `nn.CrossEntropyLoss`;
- оптимізатор `torch.optim.Adam`;
- `learning_rate=0.001`;
- 80-120 епох.

### 4. Train loop

Реалізуйте базовий train loop:

1. `model.train()`;
2. forward pass для всього `X_train_tensor`;
3. loss для всього `y_train_tensor`;
4. `optimizer.zero_grad()`;
5. `loss.backward()`;
6. `optimizer.step()`;
7. збереження train loss за епоху.

### 5. Оцінка якості

На test set порахуйте:

- `accuracy`;
- `f1_score`;
- `confusion_matrix`.

Додайте короткий висновок:

- чи навчилася модель розрізняти сорти рису;
- які помилки видно з confusion matrix;
- чи достатньо стабільним виглядає train loss.
