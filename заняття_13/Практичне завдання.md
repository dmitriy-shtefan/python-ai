## Практичне завдання

### Контекст

Команда аналізує англомовні відгуки про фільми. Перед побудовою моделі потрібно перевірити, як текст перетворюється на токени: скільки їх виходить, які токени з'являються на початку відгуків і чи є серед них шум.

### Мета

Швидко перевірити результат токенізації на готовому корпусі текстів.

#### Крок 1. Завантаження корпусу

1. Імпортуйте корпус:

```python
from nltk.corpus import movie_reviews
from nltk.tokenize import word_tokenize
import pandas as pd
```

2. Отримайте список `fileids` для позитивних і негативних відгуків:

```python
pos_ids = movie_reviews.fileids("pos")[:10]
neg_ids = movie_reviews.fileids("neg")[:10]
selected_ids = pos_ids + neg_ids
```

`fileid` уже містить категорію на початку шляху, наприклад `pos/cv000_29590.txt` або `neg/cv000_29416.txt`.
